About the DentistHub Project

Hello, my name is Alex Yusnyu and I currently study Python Web Development at SoftUni.

This is my project for my Web Development Module final assessment


DentistHub is a web application that helps clients book an appointment for their favorite dentists in just few clicks. The app connects customers with offices for an easy reservation process. The app allows 2 types of users that can register on the website: Client and Dentist.


Client users can choose from variety of dentist's offices and book an appointment.


Dentist's offices on the other hand can register on the website and can manage their reservations.



The project is developed using the following technologies:

  • For Back-end: Python, Django Framework
  • For Front-end: Custom HTML, CSS and additional components built with Bootstrap, Javascript, jQuery, LeafLet Maps Library
  • For Database: SQLite
